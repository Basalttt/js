1. Wymień cztery sposoby wyprowadzania danych w JS:
- document.write()
- alert()
- przez formularze (np. value)
- przez elementy HTML (np. innerHTML)

2. Czym jest funkcja w JS, omów przy użyciu 4 podpunktów?
- Funkcja to zbiór instrukcji wykonujących określone zadanie
- Funkcję wywołuje się po jej nazwie
- Może mieć parametry przekazywane przy wywołaniu
- Może zwracać wartość przy użyciu słowa return

3. Omów metody getElementsByName, getElementById, getElementsByTagName:

MetodaOpis
getElementById("id")Dostęp do elementu o podanym atrybucie id
getElementsByName("nazwa")Dostęp do elementów o podanym atrybucie name

4. Omów właściwości innerHTML, innerText, className:
innerHTML: pozwala odczytać lub ustawić kod HTML wewnątrz elementu.
innerText: pozwala odczytać lub ustawić tekst widoczny dla użytkownika.
className: umożliwia przypisanie klasy CSS do elementu.
